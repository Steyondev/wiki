# TranslateMC – Plugin Setup

1. Download the latest release of the plugin from [Modrinth](https://modrinth.com/plugin/translate/versions).
2. (Optional but recommended) Download the PlaceholderAPI plugin from [Modrinth](https://modrinth.com/plugin/placeholderapi).
3. Place both plugins into your server’s `plugins` folder and restart the server.
4. Create an API key on the web interface with **read** permission.
5. Open the plugin’s config file and enter your API key as well as the URL of the web interface.
6. Restart the server again, and you're ready to go!
now [install the plugin](https://steyondev.github.io/wiki/TranslateMC/Plugin/Setup/) or [use the developer java api](https://steyondev.github.io/wiki/TranslateMC/Dev/Setup/).
