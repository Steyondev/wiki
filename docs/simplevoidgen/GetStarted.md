## Installation

1. Lade das Plugin von [Modrinth](https://modrinth.com/plugin/simplevoidgen) herunter.
2. Kopiere die `.jar`-Datei in den `plugins`-Ordner deines Servers.
3. Starte den Server neu, um das Plugin zu laden.
